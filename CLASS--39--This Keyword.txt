THIS KEYWORD
==============
1.To refer current class variable
2.To refer current class methods
3.To refer current class constructors
4.To refer current class object

Note:
In Java applications,if we provide same set of variables at local and at class level and if we access that variables then JVM will give first priority for local variables,
if local variables are not available then JVM will search for that variables at class level even at class level also if that variables are not available then 
JVM will search at super class level.At all the above locations,if the specified variables are not available then compiler will rise an error.

this.var_Name                                               Ex-this.esal --->To refer current class variable
this.method_Name([param_List]);        Ex-this.m1(); ---> To refer current class methods
this([param_List]);                                       --->this(10);  ----> To refer current class constructor

02)--Use of this Keyword in Constructor to call Class variable
-------------------------------------------------------------------------------
class A
{
	int i=10;
	int j=20;
	A(int i,int j)
	{
		System.out.println(i+"    "+j);
		System.out.println(this.i+"    "+j);
	}
}
class Test
{
	public static void main(String args[])
	{
		A a=new A(30,40);
	}
}

05)--Calling of one Instance method in another Instance method
class A
{
	void m1()
	{
		System.out.println("m1-A");
		m2();                                                   //m2() method calling/Here we can use  "this.m2()"  also.

	{
		System.out.println("m2-A");
	}
	
}

{
	public static void main(String args[])
