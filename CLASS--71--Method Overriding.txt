Method Overriding 
--------------------------
Rules and Regulation for method Overriding
------------------------------------------------------------
01) To override super class method with sub class then super class method must not be declared as private.
02) To override super class method with sub class method then sub class method should have the same return type of the super class method.
03) To override super class method with sub class method then super class method must not be declared as final sub class method may or may not be final.
04) To override super class method with subclass method,sub class method must have either same scope of the super 
        class method or more scope when compared with super class method scope otherwise compiler will rise an error.
05) In method Overriding ,if we have inheritance relation between super class method throws exception and sub class method throws exception then sub class method
        throws exception must be sub class to the super class method throws exception.
06) To override super class method with subclass method subclass method should have either 
        same access privileges or weaker access privileges when compared with super class method access privileges.
